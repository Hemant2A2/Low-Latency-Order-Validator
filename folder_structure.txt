low_latency_order_validator/
├── CMakeLists.txt
│   └── Configures the C++20 build system, enforcing aggressive compiler optimizations (-O3, -march=native).
├── data/
│   └── latency_cycles.csv 
│       └── The raw hardware-cycle telemetry output generated by the pipeline, decoupled from the hot path.
├── include/
│   ├── core/
│   │   ├── messages.hpp
│   │   │   └── Defines strictly packed (1-byte aligned) C++ structs to perfectly mirror incoming binary network data.
│   │   └── spsc_queue.hpp
│   │       └── A lock-free Single-Producer Single-Consumer ring buffer using cache-line padding to defeat false sharing.
│   ├── net/
│   │   ├── udp_receiver.hpp
│   │   │   └── A kernel-bypassing UDP socket wrapper that utilizes an O_NONBLOCK spin-loop for microsecond ingress.
│   │   └── zero_copy_parser.hpp
│   │       └── Casts raw memory buffers into C++ structs using std::span, eliminating heap allocations and data copying.
│   ├── trading/
│   │   └── risk_engine.hpp
│   │       └── A stateful, branchless validation engine that tracks net positions using custom STL-allocated memory.
│   └── utils/
│       ├── huge_page_allocator.hpp
│       │   └── Interfaces with Mach kernel Superpages (macOS) and MAP_HUGETLB (Linux) to crush TLB cache misses.
│       ├── memory_pool.hpp
│       │   └── A custom Arena/Bump Allocator bridge that allows std::vector to grow dynamically without triggering OS malloc.
│       └── rdtsc_timer.hpp
│           └── Provides cross-platform access to CPU hardware registers (RDTSC / cntvct_el0) for nanosecond timestamping.
├── src/
│   ├── main.cpp
│   │   └── The grand assembly: pins threads to specific CPU cores, wires the queue, and executes the tick-to-trade hot path.
│   └── data_generator.cpp
│       └── An isolated script that blasts randomized, packed binary market data over localhost UDP to saturate the receiver.
└── benchmarks/
    └── latency_histogram.py
        └── A quantitative Python script that drops cache-warmup data, converts hardware cycles to nanoseconds, and plots p99 tail latency.
